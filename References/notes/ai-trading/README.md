# Hands-On AI Trading Notes

These notes are primarily based on the AI trading materials currently available locally: a partial full-book PDF, several chapter PDFs, and the local code repo mirror.

This folder also includes companion notes from `Advanced Portfolio Optimization` because those chapters are directly relevant to portfolio construction and AI-assisted systematic trading research.

## Files

- `available_material.md`: what book and repo material is currently present
- `advanced_sections.md`: the most technical chapters and implementation ideas
- `chapter_08_2_classic_objective_functions.md`: verbose notes on Section 8.2 from *Advanced Portfolio Optimization*
- `chapter_12_hierarchical_clustering_portfolios.md`: verbose notes on hierarchical portfolio construction from *Advanced Portfolio Optimization*
- `chapter_13_graph_theory_based_portfolios.md`: verbose notes on graph- and cluster-aware portfolio constraints from *Advanced Portfolio Optimization*
